# HubDucky

Simple webpage for serving up personal links.

```bash
yarn install --frozen-lockfile    # install
yarn dev                          # run local
yarn build                        # build prod
```

## Requires

* Node v16+
* Yarn v1.22+

## Libs

* VueJS 3
* Vite 4
* Typescript
* Vuex
* PrimeVue, PrimeIcons, PrimeFlex
* Dropbox (needed node-polyfill-webpack-plugin)
